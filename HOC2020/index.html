Additional Resources

W3 - Covering HTML, CSS, and Javascript, W3 is a great resource for learning web development. Whether you need to add an image to your page, or just figure out how to center some text, W3 is the resource for you.

Stack Overflow - A well known coding forum, with answers to virtually all programming related issues. If you can't find a solution on Stack Overflow, a solution probably doesn't exist.

Elements File - A webpage included in the template to showcase all of the included formatting options (tags) you can use to craft your website.
